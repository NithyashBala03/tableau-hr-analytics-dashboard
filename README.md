# Employee Analytics & Workforce Insights Dashboard

Workforce Insights: HR Analytics Dashboard
Project Overview

This project presents an interactive HR analytics dashboard built using Tableau to analyze workforce trends, employee demographics, salary distribution, and organizational structure.

The dashboard transforms raw HR data into clear visual insights that help organizations understand their workforce composition and support data-driven HR decision making.

Key Objectives

• Analyze employee demographics and workforce composition
• Identify salary distribution across divisions
• Track workforce growth and employee trends
• Evaluate workforce diversity metrics
• Visualize employee geographic distribution
• Provide a clear and interactive HR reporting dashboard

Dashboard Features
KPI Cards – Employee Trends

• Displays key workforce metrics and employee growth trends
• Provides a quick overview of workforce size and changes over time

Work Arrangement Analysis

• Hex-pie chart visualizing employee work types
• Categories include Remote, Hybrid, and On-site

Geographic Employee Distribution

• Hexbin map displaying employee locations across U.S. states
• Helps identify workforce concentration and geographic spread

Headcount by Division

• Heat map showing employee distribution across departments
• Enables comparison between divisions

Salary Breakdown

• Donut charts showing salary category distribution
• Stacked bar charts comparing salary ranges across divisions

Workforce Demographics

• Histogram displaying employee age distribution
• Grouped bar chart analyzing employee tenure categories

Diversity Metrics

• Icon summary cards showing gender counts
• Waffle chart visualizing gender distribution percentages

Custom Dashboard Design

• Clean dashboard layout using containers and custom visuals
• Custom legends for employment types
• Modern dashboard styling for improved readability

Tools & Technologies

• Tableau – Dashboard development and data visualization
• Microsoft Excel – Data preparation and dataset storage

Dataset Description

The dataset contains simulated HR data including:

• Employee ID
• Age
• Gender
• Salary
• Division / Department
• Work Arrangement (Remote / Hybrid / On-site)
• Employment Type
• Tenure
• Employee Location (State)

This data enables analysis of workforce demographics, salary patterns, and organizational structure.

Key Insights Generated

• Workforce distribution across departments
• Salary comparison across divisions
• Employee age and tenure patterns
• Gender diversity insights
• Employee geographic distribution across states
